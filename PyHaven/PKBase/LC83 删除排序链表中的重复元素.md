# 删除排序链表中的重复元素

### 题目描述

给定一个已排序的链表的头 `head`，删除所有重复的元素，使每个元素只出现一次。返回已排序的链表。

**示例 1：**

输入：`head = [1, 1, 2]`
输出：`[1, 2]`

**示例 2：**

输入：`head = [1, 1, 2, 3, 3]`
输出：`[1, 2, 3]`

**提示：**

- 链表中节点数目在范围 [0, 300] 内
- -100 <= Node.val <= 100
- 题目数据保证链表已经按升序排列

***

### 解法：一次遍历法

因为链表已经是排好序的，所以重复的元素一定挨在一起。我们只需要用一个指针从头到尾走一遍，遇到相邻两个节点值相同的情况，就把后面那个跳过就行。

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def deleteDuplicates(head):
    if not head or not head.next:
        return head

    curr = head
    while curr.next:
        if curr.val == curr.next.val:
            # 当前节点和下一个节点值相同，跳过下一个节点
            curr.next = curr.next.next
        else:
            # 不相同，指针往后移
            curr = curr.next

    return head
```

#### 逐步调试

用示例 2 来走一遍：`head = [1, 1, 2, 3, 3]`

**初始状态：**

链表：1 → 1 → 2 → 3 → 3，`curr` 指向第一个节点（值为 1）

**第 1 轮：curr.val=1, curr.next.val=1**

- 1 == 1？→ **相等！** 跳过下一个节点：`curr.next = curr.next.next`
- 链表变为：1 → 2 → 3 → 3
- `curr` 不移动，仍然指向值为 1 的节点

**第 2 轮：curr.val=1, curr.next.val=2**

- 1 == 2？→ 不相等，`curr` 往后移
- `curr` 指向值为 2 的节点
- 链表不变：1 → 2 → 3 → 3

**第 3 轮：curr.val=2, curr.next.val=3**

- 2 == 3？→ 不相等，`curr` 往后移
- `curr` 指向值为 3 的节点
- 链表不变：1 → 2 → 3 → 3

**第 4 轮：curr.val=3, curr.next.val=3**

- 3 == 3？→ **相等！** 跳过下一个节点：`curr.next = curr.next.next`
- 链表变为：1 → 2 → 3
- `curr` 不移动，仍然指向值为 3 的节点

**第 5 轮：curr.next 为 None**

- `curr.next` 是 None，循环结束
- 返回 `head`，最终链表：1 → 2 → 3 ✓

**测试：**

```python
def list_to_linked(lst):
    dummy = ListNode()
    curr = dummy
    for v in lst:
        curr.next = ListNode(v)
        curr = curr.next
    return dummy.next

def linked_to_list(head):
    res = []
    while head:
        res.append(head.val)
        head = head.next
    return res

print("示例1:", linked_to_list(deleteDuplicates(list_to_linked([1, 1, 2]))))
print("示例2:", linked_to_list(deleteDuplicates(list_to_linked([1, 1, 2, 3, 3]))))
```

**执行结果：**
```
示例1: [1, 2]
示例2: [1, 2, 3]
```

#### 时间复杂度分析

只需要遍历一次链表，时间复杂度是 **O(n)**。空间复杂度是 **O(1)**，只用了常数级别的额外空间。
