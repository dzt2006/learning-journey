# 爬楼梯

### 题目描述

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例 1：**

输入：`n = 2`
输出：`2`
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶

**示例 2：**

输入：`n = 3`
输出：`3`
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶

**提示：**

- 1 <= n <= 45

***

### 解法：动态规划（空间优化）

这道题其实和 [斐波那契数](./LC509%20斐波那契数.md) 本质是一样的！想想看：

- 到达第 n 阶，最后一步只有两种可能：从第 n-1 阶跨 1 步上来，或者从第 n-2 阶跨 2 步上来
- 所以到达第 n 阶的方法数 = 到达第 n-1 阶的方法数 + 到达第 n-2 阶的方法数

这不就是 F(n) = F(n-1) + F(n-2) 吗？只不过初始值不同：
- 爬 1 阶只有 1 种方法
- 爬 2 阶有 2 种方法（1+1 或 2）

```python
def climbStairs(n: int) -> int:
    if n == 1:
        return 1
    a, b = 1, 2
    for _ in range(3, n + 1):
        c = a + b
        a = b
        b = c
    return b
```

#### 逐步调试

用 `n = 5` 来走一遍：

**初始状态：** a=1（1阶的方法数）, b=2（2阶的方法数）

**第 1 轮（算3阶）：**

- c = a + b = 1 + 2 = 3
- a = b = 2, b = c = 3
- 到达3阶有 3 种方法

**第 2 轮（算4阶）：**

- c = a + b = 2 + 3 = 5
- a = b = 3, b = c = 5
- 到达4阶有 5 种方法

**第 3 轮（算5阶）：**

- c = a + b = 3 + 5 = 8
- a = b = 5, b = c = 8
- 到达5阶有 8 种方法

返回 b = 8 ✓

**测试：**

```python
print(f"n=1: {climbStairs(1)}")
print(f"n=2: {climbStairs(2)}")
print(f"n=3: {climbStairs(3)}")
print(f"n=4: {climbStairs(4)}")
print(f"n=5: {climbStairs(5)}")
```

**执行结果：**
```
n=1: 1
n=2: 2
n=3: 3
n=4: 5
n=5: 8
```

#### 时间复杂度分析

时间复杂度是 **O(n)**，只需要遍历一次。空间复杂度是 **O(1)**，只用了三个变量 a、b、c。

***

### 和斐波那契的关系

对比一下两个数列：

| n | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| 斐波那契 F(n) | 1 | 1 | 2 | 3 | 5 |
| 爬楼梯方法数 | 1 | 2 | 3 | 5 | 8 |

爬楼梯的方法数其实就是斐波那契数列往后偏移了一位——爬 n 阶的方法数 = F(n+1)。核心递推公式完全一样，只是起点不同。
